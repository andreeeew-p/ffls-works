# pinpoint

> **Superfast On-Demand NAS Search Indexer**
> 
> Pinpoint는 로컬 디스크 및 사내 공유 네트워크 드라이브(NAS) 내의 수많은 파일들을 대기 시간 없이 즉시 검색해내는 고성능 파일 검색 유틸리티입니다. 백그라운드 리소스를 낭비하지 않는 고효율 구조로 PC와 공유 서버 모두에 무리를 주지 않는 쾌적한 검색 경험을 제공합니다.

---

## ■ 왜 Pinpoint인가요?

기존의 파일 검색기들은 백그라운드에서 실시간으로 파일 시스템(MFT)을 끊임없이 감시하거나 공유 드라이브에 지속적으로 쿼리를 날려, 문서중앙화 시스템 및 NAS 서버에 상시 부하를 주곤 합니다. 

Pinpoint는 실시간 감시 방식 대신 **서버 친화적인 온디맨드 인덱싱 구조**를 적용하여 사내 공유 서버의 CPU 점유율과 네트워크 대역폭을 안전하게 보호합니다.

---

## ■ 핵심 특장점 및 기능 (Key Features)

*   **공유 서버 부하 방지 (MFT 미사용)**
    백그라운드에서 실시간으로 전체 파일 시스템을 추적하거나 감시하지 않습니다. 이로 인해 사내 공유 드라이브 및 문서 서버에 무리를 주는 불필요한 네트워크 트래픽과 디스크 I/O를 발생하지 않습니다.
*   **서버 친화적인 온디맨드 인덱싱**
    동기화가 필요한 시점에만 효율적으로 인덱싱을 갱신하도록 설계되었습니다. 대규모 네트워크 대역폭 혼잡을 방지하므로 동료들의 서버 이용 속도에 영향을 주지 않습니다.
*   **스마트 백그라운드 메모리 자동 최적화**
    프로그램 최소화 시 메모리 트리머가 자동으로 즉각 구동됩니다. 실행 중인 앱이 백그라운드에서 리소스를 독점하지 않도록 물리 RAM 사용량을 최소 수준으로 수축시켜 관리합니다.
*   **맞춤형 인덱싱 및 제외 경로 설정**
    검색하고 싶은 로컬 폴더나 네트워크 드라이브 경로만 선택적으로 지정하여 인덱스 데이터베이스를 가볍게 유지할 수 있습니다. 특정 임시 폴더나 특정 확장자는 제외 처리도 가능합니다.
*   **편리한 한글 초성 검색**
    자음 입력만으로 한글 파일명을 실시간 검색할 수 있어 업무 탐색 시간을 획기적으로 줄여줍니다 (예: \`'ㅂㄱㅅ'\` 입력 시 \`'보고서'\` 즉시 매칭).
*   **기업 정보보안 규정 준수 (Zero Telemetry)**
    네트워크를 통한 외부 데이터 수집, 트래킹, 텔레메트리 전송 기능이 일절 없는 100% 오프라인 독립 실행 구조입니다. 게다가 인덱싱된 모든 파일 경로는 로컬 컴퓨터 내에서 SQLCipher로 안전하게 암호화되어 보호되므로 까다로운 사내 정보보안 규정을 완벽하게 충족합니다.
